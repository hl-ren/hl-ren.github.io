# Online Slides System

Huilong Ren

---

## What this adds

- One URL per deck
- Keyboard navigation
- Full-screen browser presentation
- Markdown authoring
- Local images and assets per deck

---

## Deck workflow

Create one folder:

```text
slides/YYYY-MM-DD-slug/
  index.md
  images/
  videos/
  data/
```

No homepage or index edits are needed.

---

## Math support

Inline math works with \( a^2 + b^2 = c^2 \).

Block math works too:

$$
E = mc^2
$$
